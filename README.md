# Coffee Shop Website ☕

A Coffee Shop landing page built using HTML and CSS.

## About Project
This project is a modern coffee shop website design that includes:
- Navigation bar
- Hero section
- Coffee menu section
- About section
- Contact form
- Product showcase
- Blog/news section
- Reservation section
- Footer section

## Technologies Used
- HTML
- CSS
- Font Awesome Icons

## Features
- Clean and attractive UI design
- Coffee themed layout
- Responsive design for mobile and desktop
- Modern sections and cards
